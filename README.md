# Ng Course Recipe Book

The project is built with Angular 9. It allows users to create, update, delete and load recipes and add/delete ingredients for the recipes. 

The application consists of many modules, like App module, core module, shared modules, and a few feature modules. It uses strategies such as lazy loading and preloading to improve performance.

The backend of the project is Firebase. The application allows users to sign up, log in and log out using email and password.

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 7.3.5.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
